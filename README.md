# blog 博客
## 第一次启动工程

npm install 下载后端开发需要的依赖包

bower installer 下载前端开发需要的依赖包

bower-installer 将前端需要的依赖包根据配置映射到项目的build_bower目录

gulp build-script 编译react的jsx文件，生成对应的js文件

gulp sass   编译sass的*.scss文件，生成对应的css文件

node server 启动项目

## 开发监听

gulp js-watch

gulp sass-watch
